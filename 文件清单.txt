========================================
    如何使用这些文件
========================================

下载整个项目:
  位置: /tablet_ime/
  
  方法 1: 使用 Git
    cd /tablet_ime
    tar -czf tablet_ime.tar.gz .
    # 下载 tablet_ime.tar.gz

  方法 2: 逐个下载
    下载所有文件和文件夹

构建 APK:
  Windows: 双击 /tablet_ime/一键构建APK.bat
  Mac/Linux: 运行 /tablet_ime/build_apk.sh
  GitHub Actions: 上传到 GitHub 自动构建

阅读文档:
  首先看: /tablet_ime/快速开始.txt
  详细教程: /tablet_ime/如何创建APK.md
  完整指南: /tablet_ime/BUILD_GUIDE.md
