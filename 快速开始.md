# 快速创建 APK - 最简单步骤

## 第一步：创建 GitHub 账号和仓库

1. **访问 GitHub**
   - 打开浏览器，访问: https://github.com
   - 注册账号（如果还没有）

2. **创建新仓库**
   - 登录后，点击右上角 "+" 
   - 点击 "New repository"
   - 仓库名: `tablet-ime` (任意名称都可以)
   - 选择 "Public"
   - 点击 "Create repository"

## 第二步：上传代码

**最简单方法 - 直接拖拽上传：**

1. 在创建的仓库页面，点击 "uploading an existing file"
2. 打开文件夹 `tablet_ime`
3. 选择所有文件，拖拽到浏览器
4. 填写提交信息: "Initial commit"
5. 点击 "Commit changes"

**或使用 Git 命令（如果熟悉）：**
```bash
cd tablet_ime
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/YOUR_USERNAME/tablet-ime.git
git branch -M main
git push -u origin main main
```

## 第三步：自动构建 APK

代码上传后，GitHub 会自动开始构建（约 5-10 分钟）

1. 点击仓库的 "Actions" 标签
2. 等待 "Build APK" 完成变为绿色 ✓
3. 点击最新的构建任务

## 第四步：下载 APK

1. 在构建任务页面，滚动到底部
2. 找到 "Artifacts" 部分
3. 点击 `app-release-arm64.apk` 下载
4. 保存到电脑

## 第五步：安装到设备

1. 将 APK 文件发送到手机/平板
   - 通过微信/邮件/数据线
2. 在设备上打开文件管理器
3. 找到 APK 文件，点击安装
4. 允许安装未知来源应用
5. 完成安装

## 第六步：启用输入法

1. 打开设备 **设置**
2. 进入 **语言和输入法**
3. 找到 **Tablet IME**，开启开关
4. 点击 **当前输入法**，选择 **Tablet IME**

## 完成！

现在你就可以使用平板输入法了！
