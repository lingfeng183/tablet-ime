====================================================================
    如何创建 APK 文件（3种方法）
====================================================================

方法 1：Windows 电脑（最简单）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 下载整个 tablet_ime 文件夹
2. 双击运行：一键构建APK.bat
3. 等待 5-10 分钟
4. APK 自动生成到：build\app\outputs\flutter-apk\app-release.apk

前提：
- 电脑已安装 Flutter
- 如果没有，访问：https://flutter.dev/docs/get-started/install/windows

方法 2：Mac/Linux 电脑
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 打开终端
2. cd 到 tablet_ime 目录
3. 运行：
   chmod +x build_apk.sh
   ./build_apk.sh
4. 等待 5-10 分钟
5. APK 生成到：build/app/outputs/flutter-apk/app-release.apk

方法 3：GitHub Actions（不需要安装任何软件）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 注册 GitHub 账号：https://github.com/signup
2. 创建新仓库：tablet-ime
3. 上传代码（拖拽上传所有文件）
4. 等待自动构建（5-10 分钟）
5. 在 Actions 页面下载 APK

详细步骤：见 快速开始.txt
